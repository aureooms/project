Swiss Army KnifE
================


***sake*** is a module based tool


usage :

	$ <module> <action> <*arg>


currently available modules and actions :

  - help
  - git
  - site
  - sublime


examples :

  - retrieve info on git repos located under *.* and *workspace*

	$ git info . workspace

  - list actions available in module *site*

	$ help list site
