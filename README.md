Swiss Army KnifE
================


***sake*** is a module based tool


usage :

	$ <module> <action> [<*arg>]


currently available modules and actions :

  - help
  - git
  - site
  - sublime


examples :

  - retrieve info on git repos located under <path1>, <path2>, ...

	$ git info <path1> <path2> ...

  - list actions available in module <m>

	$ help list <m>
